Qugga 使用

QUAGGA 说明

##　quagga　安装

  • 前提开启ip forward转发

    1
2

    echo "net.ipv4.ip_forward = 1" >> /etc/sysctl.conf
sysctl -p 　生效

  • apt　install

    1

    apt-get install quagga

  • 配置/etc/quagga/daemons

    1
2
3
4
5
6
7
8

    zebra=yes
bgpd=no
ospfd=yes
ospf6d=no
ripd=no
ripngd=no
isisd=no
babeld=no

  • 配置ospf

    1
2
3
4
5

    cp /usr/share/doc/quagga/examples/zebra.conf.sample /etc/quagga/zebra.conf
cp /usr/share/doc/quagga/examples/ospfd.conf.sample /etc/quagga/ospfd.conf
  
chown quagga.quaggavty /etc/quagga/*.conf
chmod 640 /etc/quagga/*.conf

  • 启动服务

    1

    systemctl restart quagga
    • 优化:
    1
2
3

    echo VTYSH_PAGER=more > /etc/environment 
or
export VTYSH_PAGER=more    到~/.bashrc

  • 验证服务

    1
2

 no link-detect

 no link-detect
!
interface lo
 no link-detect
!
router ospf
 ospf router-id 172.16.237.10
 log-adjacency-changes
 auto-cost reference-bandwidth 1000
 network 172.16.3.100/32 area 0.0.0.0　　＃发布vip
 network 172.16.237.0/24 area 0.0.0.0        # 发布本网段路由

ip forwarding
!

!
