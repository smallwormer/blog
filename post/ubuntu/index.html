  • 在使用vmware 的时候出现了vmnet-natd: RTM_DELADDR: index 然后工作机就出现了ipaddress 消失的问题，

    需要重新dhcp 才可以重新获取IP, 即便是配置了静态IP 也是一样

    1

    https://www.nikhef.nl/~janjust/vmnet/   改变vmware net 的模块，然后重新加载完成

    1. 找到模块文件/usr/lib/vmware/modules/source/vmnet.tar

    2. 解压并更改文件

      cd     /usr/lib/vmware/modules/source/
tar -xf vmnet.tar
vim vmware-only/userif.c
     
     userIf = (VNetUserIF *)port->jack.private;
      hubJack = port->jack.peer;
      
  +   /* never send link down events */

         return -EINVAL;
      }

    3. 压缩tar cf vmnet.tar vmware-only/

    4. 重新加载模块

      1

      /usr/bin/vmware-modconfig --console --install-all

    5. 重启vmware 服务

      1

      systemctl restart vmware
